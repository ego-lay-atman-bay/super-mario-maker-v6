Variables

See other tables

These variables are used by the command block.

Note: These are NOT case sensative.

Variable Info
blueOnOff Regular on/off block state. 1 = on, 0 = off.
greenOnOff Green/yellow on/off block state. 1 = on, 0 = off.
pSwitch P-switch state. 1 = on, 0 = off
conditionCleared Clear condition cleared? 1 = yes, 0 = no
conditionCounter What is shown in the variable next to the clear condition image
conditionValue Clear condition value, amount of stuff you need to collect
cloneBlock How many block clones are loaded
cloneEntity How many entities loaded
startTime What the timer starts out with
maxTime Current timer limit
timer Current timer value
sharelvl Tests if the user is sharing the level. "yes" or "no".
yoshiDamage Tests if mario has been damaged while on yoshi. 1 = true, 0 = false.
action Current game action. Note: value is a string (but 0 = playing)
startCharacter What character you start with.
gameMode "create" or "play"
music Current playing music.
startPowerup What powerup mario starts with.
xmax X max of the level.
xstart X position mario starts at. Note: this is changed by checkpoints.
ymax Y max of the level.
ystart Y position mario starts at. Note: this is changed by checkpoints.
marioDamage Checks if mario is abe to be damaged. 1 = false, 0 = true.
marioCrouch Checks if mario is crouching.
marioCostume Current character/costume mario is playing as.
marioDirection Current direction mario is looking. "right" = →, "left" = ←.
marioHitx Mario's hitbox x. mariox + marioHitx = mario's right.
marioHity Mario's hitbox y. marioy + marioHity = mario's top.
marioHold Whether mario is holding an item. 1 = holding, 0 = not holding.
marioHoldItem What item is is mario's hand. Value is a string.
marioJumpHight Mario's max jump height.
cloudNum How many clounds mario has.
keyNum How many keys mario has.
onYoshi Checks whether mario is on yoshi.
powerup Mairo's current powerup.
marioSpeed Current speed mario is moving at.
marioSpin Checks if mario is spinning. 1 = true, 0 = false.
marioFall Checks mario's current fall/jump state. -1 = falling, 0 = standing, 1 = jumping.
usePowerup Checks whether mario is using a powerup (right now, only fire mario effects this)
mariox Mairo's x position. Decimal value.
marioy Mario's y position. Decimal value.
marioVol Mario's current velocity.
startMusic Starting music.
airJump Whether mario you can do a mid-air spin jump. 1 = true, 0 = false.
theme Current theme.
bgc Current background color.
scrollx Current camera x position.
scrolly Current camera y position.
scrollxMax Scroll x max.
scrollyMax Scroll y max.
coins Amount of coins you currently have.
fps Current fps (the fps variable).
tick Current frame.
totalClones Total clones on screen.
lives Amount of live you have left. Only used in 100 mario challenge.
projectVer Project version the level was created in. 0 = original project.
blockx Block x.
blocky Block y.
animation Current animation type, 1 = animate.
fullid Full id.
id Id (most likeley 61)
TRUE TRUE
FALSE FALSE
